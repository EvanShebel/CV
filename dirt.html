BEV Motorcycle Frame

Battery Electric Dirt Bike Frame


The first attempt at modeling

The latest iteration


When teaching myself to model this frame I found it difficult to model organic shapes solely with feature modeling. See my first attempt above. My solution was to learn surfacing. As far as the surfacing tools I used, I made use of nearly every tool in the stock surfacing toolbar. Additionally I made use of the other tools in the drop down menus, tools such as fit spline which I used to create uniform guide profiles to avoid tangent lines on what should be class A surfaces.

Also I realized that it was necessary that I teach myself a design method, such as top down design. My preferred design method is to create a multibody part file and save the separate parts to new child parts.

Having spent a lot of time modeling in SolidWorks I made some customizations to the user interface to speed up modeling. I mapped frequently used features to hot keys and also made use of the mouse gestures in different enviroments.
